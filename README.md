# Image Search (serpapi)

- UIKit
- MVC + Service
- No Storyboard, No Snapkit
- SwiftLint (CocoaPods)
- Image Cache
- Prev/Next image scroll on full size image VC
- Web View with original source of image  
  
Please, run pod install in terminal, before run app in Xcode.
